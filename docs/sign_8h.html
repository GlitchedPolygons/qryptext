Signs a given byte array (message) using Falcon-1024 (which within OQS uses SHAKE256 hashing internally).

Parameters
dataThe data to sign.
data_lengthLength of the data array.
output_bufferWhere to write the signature into.
output_buffer_sizeHow big the signature output buffer is. Make sure to allocate at least OQS_SIG_falcon_1024_length_signature bytes!
output_lengthHow many bytes were written into the output buffer.
output_base64Should the output signature bytes be base64-encoded for you? Pass a non-zero value for true. If you pass true, make sure to allocate at least qryptext_calc_base64_length(OQS_SIG_falcon_1024_length_signature) bytes! If you pass false, the raw signature bytes are written into the output_buffer!
secret_falcon1024_keyThe Falcon-1024 secret key to use for signing.
Returns
0 on success; error codes as defined in constants.h otherwise.
