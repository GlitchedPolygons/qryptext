Encrypts a given byte array of data using AES256-GCM + Kyber1024.

Parameters
dataThe data to encrypt.
data_lengthLength of the data array.
output_bufferWhere to write the encrypted ciphertext into. Make sure that this is allocated sufficiently big! If you're unsure about how much to allocate, you can use util.h's qryptext_calc_encryption_output_length() function.
output_buffer_sizeHow big the output buffer is (use qryptext_calc_encryption_output_length() for allocation size guideline).
output_lengthWhere to write the number of bytes written to the output buffer into (will be left untouched in case of a failure).
output_base64Should the encrypted output bytes be base64-encoded for easy transmission over e.g. email? Pass a non-zero value for true. If you decide to base64-encode the encrypted data buffer, please be aware that a NUL-terminator is appended at the end to allow usage as a C-string but it will not be counted in output_length.
public_kyber1024_keyThe Kyber1024 public key with which to encrypt the AES key.
Returns
The status code: 0 if encryption succeeded, all other status codes can be found inside the various qryptext header files.
