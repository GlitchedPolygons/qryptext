Decrypts a given byte array of data that was encrypted using qryptext_encrypt().

Parameters
encrypted_dataThe data to decrypt.
encrypted_data_lengthLength of the data array.
encrypted_data_base64Is the encrypted data a base64-encoded string? Pass a non-zero value for true.
output_bufferWhere to write the decrypted data into (make sure that this is allocated sufficiently big!).
output_buffer_sizeHow big the output buffer is (usually, when unsure, allocate the same size as the encrypted data length; that's guaranteed to be sufficiently big).
output_lengthWhere to write the number of bytes written to the output buffer into (will be left untouched in case of a failure).
secret_kyber1024_keyThe Kyber1024 private key with which to decrypt the AES key.
Returns
The status code: 0 for success, all other status codes can be found inside the various qryptext header files.
